## 使用流程

```bash
git clone https://github.com/Silence-dream/chrome-Extension-A-SOUL.git
cd chrome-Extension-A-SOUL
yarn;yarn build
# 生成打包文件在 dist 文件夹内
# 进入 chrome://extensions 打开开发者模式
# 点击加载已解压的扩展程序选择 dist 文件夹即可成功载入
```
